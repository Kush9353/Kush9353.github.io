AI MIPS Pipeline Scheduler

Designed an intelligent instruction scheduler for a MIPS pipeline using a Q-learning (Reinforcement Learning) algorithm. This AI-based approach dynamically reorders instructions to minimize stalls, achieving an 18.7% increase in IPC and a 12.8% reduction in energy consumption.

Built a wearable IoT system using an RP2040 microcontroller to continuously monitor an asthma patient's vital signs (SpO2, heart rate, temp). Data is sent to a Firebase cloud database for real-time visualization on a web/mobile dashboard via Node-RED.
